This is an open source project that is a nostr agent that can communicate with bitcoin wallets about the market and stuffs and sometime via nostr

Current 

**Current features:**

- [ ] Show keys
- [ ] Store agent owner public key
- [ ] Store agent owner private key
- [ ] Store agent owner public key
- [ ] Fetch Private DMs between agent and owner from nostr

how to run the project:

1. clone the repository
2. run `npm install` to install the dependencies
3. run `npm run start` to start the project
